  2. TC Electronic PolyTune 3

Song-Specific Deployment

  • Tuning: Eb standard (Eb–Ab–Db–Gb–Bb–Eb). Matches Hendrix studio tuning — KWS plays it in B-natural shapes against Eb tuning.
  • Pickups: Bridge for the main riff. Neck for the funk-break "stand up next to a mountain" section.
  • Intro / main riff: Wah ON. Drive stack OFF for the dry-clean opening; kick TS808 in as the riff settles. Phase 90 OFF.
  • Funk break: Engage Phase 90 (script). Neck pickup. Wah on the rhythmic chord stabs.
  • Solos: Stack TS808 + Duellist Heavy Hand. Bridge pickup. The Blues Deluxe should be audibly grinding.
  • Octavia moment: One small octave-up section — skip if you don't have an octave pedal patched in.
