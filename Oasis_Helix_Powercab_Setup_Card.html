Why it sounded like trash before — double-cabbing

A Helix Cab/IR block running at the same time as a Powercab speaker model processes the speaker twice → boxy, dark, dull, "digital." Line 6's own rule: to use Powercab speaker models, disable cab blocks in the modeler.

The one rule that fixes it permanently

Powercab → FLAT mode, forever. All tone lives in the Helix. Now the Helix cab block is the only speaker sim (correct), there's no two-box state to manage, and double-cabbing is structurally impossible. Set the Powercab once and never touch it again.

A · One-Time Setup (do once, never again)

  1. Cable: one 110Ω XLR — Helix L6 LINK OUT → Powercab L6 LINK IN (carries audio + control).
  2. Firmware: Helix ≥ 2.80, Powercab ≥ 2.00 (older = no L6 LINK control).
  3. Powercab → FLAT mode, master ~12 o'clock. Last time you touch the Powercab.
  4. Helix Global → Ins/Outs: In-Z = Auto; Pad off (humbuckers — only on if input clips).
  5. Helix Global → Global EQ → OFF. A forgotten Global EQ silently wrecks every patch.
  6. Loudness is set at the Powercab master, not by cranking Helix output. Output meters peak, never clip.

B · The Dial-In Method (follow the order — never freelance)

  1. Start from an empty preset (never edit a factory patch — full of blocks you'll fight).
  2. Place one Amp+Cab block (combined = fewest blocks). Cab stays IN because Powercab is on Flat.
  3. Dial the amp naked — Gain → Bass/Mid/Treble → Presence → Master. Get it right before anything touches it. Keep Master up (~7) or it's all preamp fizz.
  4. Set the cab High Cut to 6–8 kHz — kills the digital ice-pick. This one setting fixes most "harsh" complaints.
  5. Then add drive. Then mod/time. Set levels LAST.

No literal JCM900 in Helix

The 90% floor — 5 minutes, zero menu-diving

Do section A once. Then build a 2-block patch: Scream 808 → Brit 2204 (Amp+Cab, High Cut 7 kHz, Master 7). That's 90% of Be Here Now. Everything beyond those two blocks is polish.

Where the Helix actually earns its keep: the Morning Glory AC30 + Bluesbreaker blend — two amps summed live, which analog can't do without two amps + an ABY + mic'ing both. For single-amp Be Here Now, your analog rig is honestly fine.
